📊 Analyzing Amazon Prime User Behavior

This project analyzes the Amazon Customer Behavior Survey dataset from Kaggle to understand user demographics, shopping patterns, and interaction with recommendation systems. Using Python for data preprocessing and visualization, the study explores correlations between user satisfaction, rating accuracy, search behavior, and feature usage, offering actionable insights for improving e-commerce experiences.


🛠️ Key Components:

  • Data Preparation: Cleaned timestamps and normalized complex fields (e.g., split multi-category columns)
  • User Profiling: Analyzed distributions by age, gender, browsing and purchase frequency
  • Behavioral Insights: Investigated search methods, recommendation usefulness, and save-for-later behaviors
  • Satisfaction Analysis: Cross-analyzed satisfaction scores with rating accuracy
  • Trend Discovery: Explored time-based changes in unique user counts real-world relevance.

🧰 Tools & Technologies：

  • Language: Python
  • Analysis & Visualization: Pandas, NumPy, Matplotlib, Seaborn
