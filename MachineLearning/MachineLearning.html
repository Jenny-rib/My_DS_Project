🎯 Click-Through Rate (CTR) Prediction Model & Demo

This project utilizes the large-scale AVAZU dataset to build a Random Forest model for predicting click-through rates (CTR). It also includes an interactive Streamlit demo for exploring the dataset, analyzing features, and simulating CTR predictions based on user inputs. visualized user data to uncover behavioral patterns related to age, content preferences, devices, and subscription timing. Interpreted the underlying causes of user behavior and identified future trends. Illustrated data-driven business insights through storytelling cases to enhance real-world relevance.


🛠️ Key Components

  • Data Preparation: Used the AVAZU click log dataset (~4 million rows, 25 features)
  • Feature Engineering: Combined time features, created ratio features, and applied log transformation
  • Model Training: Trained a Random Forest classification model to predict click behavior and then evaluated feature importance and improved generalization
  • Web Demo: Developed a Streamlit app with data exploration and prediction simulation

🧰 Tools & Technologies：

  • Language: Python
  • Visualization & Modeling: Pandas, NumPy, Matplotlib, Scikit-learn (Random Forest)
  • Web Demo: Streamlit(interactive web interface)
