🧠 Retrieval-Augmented Generation (RAG) System for Sustainability Insights

This project implements a RAG system that allows users to ask questions about the Future of Sustainability 2025 whitepaper and receive AI-generated answers based strictly on the document content. It combines semantic retrieval with generative AI to support accurate and contextual responses.


🛠️ Key Features & Workflow:

  • PDF Document Loader: Loads the sustainability whitepaper into memory.
  • Text Chunking & Embedding: Splits the document and encodes it using BAAI/bge-base-en-v1.5.
  • Milvus Vector Database: Stores embeddings and enables similarity search to retrieve top-k relevant context.
  • LLM Answer Generation: Uses Google’s flan-t5-large model to generate answers from retrieved context.
  • Strict Prompting Logic: Ensures responses are grounded only in verified document content.
  • Streamlit Web Interface: Users can interact with the system in a clean, browser-based demo.

🧰 Tools & Technologies：

  • Language Models: flan-t5-large (Hugging Face)
  • Embedding Models: BAAI/bge-base-en-v1.5
  • Embedding Models: Vector Store: Milvus
  • Interface: Streamlit
  • Supporting libraries: Langchain, SentenceTransformers, PyPDFLoader, etc.
